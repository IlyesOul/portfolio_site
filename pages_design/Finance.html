program output

program description

this is Theta Terminal. it has various features which all serve to suggest trading strategies for different stocks. this includes means of research, prediction, and visualization. I plan to automate this proccess once I integrate most of the ideas I have into it.

this project is a work in progress, as I'm always getting new ideas and trying to program them. The version you will see in the Github is simply a version of the project, and there will be no "fluid" input/UI proccess. I'm also working on this project with a couple of friends, who are integrating classic financial algorithms/indicators into the program while I focus on the ML aspect

technologies used

  1. Yahoo! Finance API
  2. JSON API Queries
  3. matplotlib Graphing
  4. Random Forest Ensembles
  5. Numpy & Pandas
  6. Abstract Methods
