Program Description

The program will begin by reading a predetermined 2D Array of parents and their children in a family. It will then construct a linked list with two pointers (NOT DOUBLY-LINKED) that will retain the generational relationships between all of the people. The user can then acquire a list of any person's relatives.

While a Binary/RB tree seems more intuitive, the underlying linked list utilizes two seperate pointers and doesn't need to be sorted.

The average time complexity is O(N)

Technologies Used

  1. Linked Lists (for storing peices)
  2. 2D Arrays
  3. Recursion
