Program Description

Welcome to ... the Space Race! Not the Soviet-American one, but the JavaScript one.

This is a fun two-player game that I made in JavaScript where each user is trying to get to the opposite side of the screen as many times as possible to beat the other player while doing their best to dodge the incoming asteroids in under a minute (so maybe it IS a little like the Cold War one)

Technologies Used

  1. JavaScript
  2. p5.js library
  3. Randomization
