Program Description

This is "Sling 'em and Stack 'em! The player is given a yellow block (depicted in output image) that they must use to knock down the stack of cubes under the cannon while defending their own stack from the cannon's attacks.

I originally made this game in the image of Angry Birds, but I made it to be more engaging by having a computer-opponent that actively tries to knock your stack down while you try to hit its stack

Technologies Used

  1. JavaScript
  2. p5.js library
  3. matter.js Physics library
